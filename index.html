Surface Wind speed retrieval using ML on CYGNSS GNSS-R Data

The project focuses on the development Machine learning models for the retrieval of ocienic winds using GNSS-R data from CYGNSS cluster of satellites. We used various architectures of neural networks and in one of the instance explored powerful technique of Principal Component Analysis on the CYGNSS L1 dataset, with the ECMWF wind dataset as a reference.

Uncertainty in NILM

The project aims to predict the uncertainty in deep learning models of Non-Intrusive Load Monitoring (NILM) models (Seq2Seq and Seq2Point) using MC dropout and deep ensemble methods.

Decision Tree from scratch

A python project made just using Pandas and NumPy library which provides functionalities similar to Scikit-learn implementation of decision trees to train, test and evaluate decision tree models on any data.
